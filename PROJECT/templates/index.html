ABOUT CYBERSHIELD AI

CyberShield AI is an intelligent network security platform that detects cyber threats using Graph Neural Networks.

The system converts network traffic into graph structures where:

  • IP addresses are represented as graph nodes.
  • Communication links are represented as graph edges.
  • Traffic features such as packet size, protocol, and duration are used as node attributes.

A trained Graph Convolutional Network (GCN) analyzes network behaviour using message passing and graph feature learning to classify traffic into:

  • 🟢 Normal Traffic
  • 🟠 Suspicious Activity
  • 🔴 Attack Traffic

The platform supports:

  • Dataset based network analysis
  • Real-time packet monitoring
  • GNN threat prediction
  • Interactive graph visualization
  • Adaptive security alerts
